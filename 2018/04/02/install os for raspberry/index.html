为树莓派安装操作系统"RASPBIAN"

发表于 2018-04-02

快速开始

现在

验证一下官网的sha-256

解压img文件到/home/user/目录下

查找SD卡

先将SD卡全部卸载

利用dd命令将镜像写入SD卡

bs可以修改大不要太大

实在想看写镜像速度情况,打开一个新的terminal

将SD卡插入树莓派中,通电插上显示器就ok了

*使用dd 命令可以覆写任何分区。如果你在指令中指定了错误的设备，你有可能删除你的操作系统。

*使用 dd 写入，写出的操作可以用于备份SD卡里操作系统和数据文件，随时备份，随时恢复，而且内容完全一样。
